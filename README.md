# MagClear
Интернет магазин. MVC. Слоёная архитектура. CRUD

**Стэк:**
- Трёхслойная архитектура: *WEB->BL->DAL*
- MVC
- Identity
- EF core
- MySql
- Docker/ docker-compose

**Функционал:**
- Аутентефакция\Авторазиция на JWT
- Админка
- CRUD товаров
- Корзина
- Система оповищений (email)
- Верификация аккаунта
